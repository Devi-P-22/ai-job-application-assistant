#  AI Job Application Assistant

##  Overview
This is an AI-powered web application that helps users improve resumes, get career advice, and prepare for job applications using intelligent responses.

##  Problem
Many students struggle with resume building and career guidance.

##  Solution
This project provides an AI assistant that analyzes user queries and gives real-time suggestions.

##  Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- AI API: OpenRouter
- Deployment:
  - Backend: Render
  - Frontend: Vercel

##  Features
- Real-time AI chat
- Resume improvement suggestions
- Career guidance
- Cloud deployed system

##  How it Works
1. User enters query
2. Frontend sends request to backend
3. Backend calls AI API
4. AI response is returned and displayed
