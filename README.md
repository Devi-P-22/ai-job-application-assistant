# AI Job Application Assistant 

##  Overview
AI-powered web app that helps users with career guidance.

## Tech Stack
- React.js
- Node.js
- Express
- AI API (OpenRouter)

##  Features
- Chat-based UI
- Real-time AI response
- Agentic workflow

##  How to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start
